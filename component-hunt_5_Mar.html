1. Website Chosen: YouTube

6. Key Observations

  • Reusability: VideoCard, NavItem, and Chip are repeated many times with different props — perfect for components.
  • Composition: VideoCard is itself composed of Thumbnail, VideoInfo, and ChannelAvatar.
  • State lifting: Search query and active category live in a parent and filter the videos array passed down to the grid.
  • Single Responsibility: Each component handles one concern — SearchBar handles input, VideoGrid handles layout, VideoCard handles display.
